Mac连接Windows在工作组里分享的打印机

Apr 12, 2017 | 其他

昨天实验室的学霸把打印机折腾好了，以后打论文终于方便多了，但还是有个问题只有Windows可以很方便的加入Windows创建的家庭组，然而Mac并不能加入，并且我的Mac和实验室的打印机并不在同一子网下，在网上看了好多帖子，终于找到了解决方案。此处所用打印机型号为HP LaserJet 1020 Plus，Toshiba打印机可以参照知乎上的这个问题。

1. 驱动安装

Mac在系统中自带了一些打印机驱动，但都不是具体到某个型号，只是写着Series，所以有些打印机在直接插到Mac上时虽然可以正常添加，但是并不能正常打印，这时就需要去打印机官网下载驱动，并且使用Windows分享的打印机设备同样需要安装驱动。我在苹果的support官网找到了惠普打印机驱动的打包下载，链接这里贴出来方便大家–Mac上惠普打印机驱动打包下载。

2. 打开系统偏好设置（Preference）中的打印机与扫描仪

3. 添加打印机

4. 使用高级选项卡配置打印机信息

如果打印机名称中有空格记得要用“%20”代替（不包含双引号），如果有中文….那就查一下中文对应的utf-8编码然后前面加上%

5. 大功告成！

选择完驱动就可以点添加(Add)了，这时会转一会儿菊花，系统正在配置打印机，配置完成后就可以正常使用啦～～～
